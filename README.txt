1. INSTALLATION INSTRUCTIONS

To build and run the road optimiser software, the following libraries need to be installed:
- Boost http://boost.org/
- Eigen http://eigen.tuxfamily.org/
- LibIGL http://libigl.github.io/libigl/
- CTPL https://github.com/vit-vit/CTPL.git
- GDAL http://www.gdal.org/index.html/
# - Dakota https://dakota.sandia.gov/
#- SAGA http://http://saga-gis.org/
#- CSIRO Workspace https://research.csiro.au/workspace/
#- QuantLib http://quantlib.org/index.shtml
#- Marxan http://marxan.net

In addition, the following libraries are included in the source code for this software under the GPL2+ license. They do not need to be installed but are mentioned here as they are incorporated in the repository.
- ALGLib http://alglib.net
